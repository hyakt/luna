luna
===
Simple fish shell prompt forked by sol.

<docs/demo.gif>

Features
---
- Git
  - Clean
  - Dirty / Touched
  - Staged
  - Staged + Dirty
  - Stashed
  - Unpushed commits (ahead)
  - Unpulled commits (behind)
  - Unpulled and unpushed commits (diverged)
  - Detached HEAD
  - Branch name

Install
---
```fish
fisher install hyakt/luna
```
